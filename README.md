# 📊 Music Streaming Data Analysis

📌 Overview

This project analyzes user behavior in a music streaming platform using SQL, Python, and Excel-based datasets (ODS/XLSX).

The goal is to extract actionable insights to improve user engagement, retention, and Premium conversion rates.

## 🎯 The analysis focuses on:


- User behavior patterns
- Subscription type differences
- Device and country segmentation
- Conversion optimization strategies
  

## 🗂️ Dataset

The dataset represents user activity on a music streaming platform.

Columns:

- user_id → user identifier  
- age → user age  
- country → country  
- subscription_type → subscription type  
- device → device type used  
- daily_minutes → daily listening time  
- favorite_genre → preferred music genre  
- skip_rate → song skipping rate  
- ads_clicked → ad click activity  
- churned → whether the user left the platform  

## 🛠️ Tools & Technologies

- Excel (ODS/XLSX for raw dataset storage)
- SQL (PostgreSQL / SQLite) 
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Colab (for analysis & visualization)


## 🧠 Analyses Performed

The following questions were explored in this project:

- Do free and premium users have different average listening times?
- Do churned users have higher skip rates?
- Which countries have the highest user density?
- How does user behavior differ across device types?
- Which music genres have higher skip rates?
- Are there missing values or outliers in the dataset?

## 📈 Key Insights

- Premium users show significantly higher engagement levels
- Mobile users dominate overall streaming activity
- Certain countries exhibit higher conversion potential
- Low engagement users are strong churn candidates

## 🚀 Business Recommendations

- Develop personalized recommendation systems to increase Premium conversion
- Target high-usage Free users with subscription incentives
- Segment users for personalized marketing campaigns


## 📌 Project Goal

- To transform raw streaming data into actionable business insights that improve user retention, engagement, and revenue growth.


## 🚀 Conclusion

This project demonstrates how different tools can be combined in a data analysis workflow.

By using Excel for exploration, SQL for querying, and Python for analysis and visualization, meaningful insights about user behavior were generated.
